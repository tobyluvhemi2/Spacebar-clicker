How to Use

Start

Begin the stopwatch timer. Click again to continue from current time.

Pause

Temporarily stop the timer without resetting the time.

Resume

Continue timing from where you paused.

Reset

Reset the timer to zero and clear all lap times.

Lap Time

Record the current time while the stopwatch continues running.

Keyboard Shortcuts

Space: Start/Pause • L: Lap • R: Reset
